# calendar
Print the Calendar of the year/month

## Usage
  main.py [OPTIONS]

  :param year: int :param month: int

Options:
  -y, --year INTEGER   Enter the calendar year  [required]
  -m, --month INTEGER  Enter the calendar month. Format 1-12
  --help               Show this message and exit.

### Example:

main.py -y 2021

main.py -y 2021 -m 12
